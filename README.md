
# popular_movies

A Flutter project.
Simple movie viewer for the most popular movies.

Functionalities:
- display popular movies
- display movie detail - poster and description
- filter movies by title
- display movie detail - vote average

Tech Stack:
- Dart
- JSON Serializable
- DIO
- Retrofit
- RXDart
